# Azure Data Pipeline Demo

## Goal
Build an end-to-end Azure data ingestion and reporting pipeline that demonstrates practical Azure skills for entry-level data/cloud roles.

## Planned Architecture
- Source data (CSV)
- Azure Blob Storage (raw landing zone)
- Azure SQL Database (structured tables)
- Azure Data Factory (ingestion + load)
- SQL queries for reporting/validation
